# Usage
<!-- usage -->
```sh-session
$ npm install -g @cooknjs/whisker
$ whisker COMMAND
running command...
$ whisker (-v|--version|version)
@cooknjs/whisker/0.1.0 linux-x64 node-v14.17.5
$ whisker --help [COMMAND]
USAGE
  $ whisker COMMAND
...
```
